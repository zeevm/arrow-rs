contains the concatenation of the histograms of all pages

§valid: bool

Is the information in the builder valid?

Set to false if any entry in the page doesn’t have statistics for some reason, so statistics for that page won’t be written to the file. This might happen if the page is entirely null, or is a floating point column without any non-nan values e.g. https://github.com/apache/parquet-format/pull/196
