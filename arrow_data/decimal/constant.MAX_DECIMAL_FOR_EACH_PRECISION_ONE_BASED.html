Expand description

MAX_DECIMAL_FOR_EACH_PRECISION_ONE_BASED[p] holds the maximum i128 value that can be stored in [arrow_schema::DataType::Decimal128] value of precision p. The first element is unused and is inserted so that we can look up using precision as the index without the need to subtract 1 first.
